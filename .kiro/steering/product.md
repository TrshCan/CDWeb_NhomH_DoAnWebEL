# Product Overview

This is a fullstack social survey platform that combines social networking features with survey/quiz management capabilities.

## Core Features

- **Social Feed**: Users can create posts, share content, like, comment, and follow other users
- **Survey Management**: Create, edit, duplicate, and manage surveys and quizzes
- **Survey Participation**: Users can join surveys, submit answers, and view results
- **Group Management**: Create and manage groups with posts and members
- **User Management**: Profile management, authentication, role-based permissions
- **Admin Dashboard**: Comprehensive admin panel for managing users, badges, events, deadlines, and surveys
- **Analytics**: Survey overview with statistics, raw data export, and response details

## User Roles

- **Admin**: Full system access and management capabilities
- **Lecturer**: Can create and manage surveys/quizzes, view responses
- **Student**: Can participate in surveys, view public content, manage own profile

## Key Entities

- Surveys/Quizzes with questions and options
- Posts with media, likes, shares, and comments
- Groups with members and group-specific posts
- Events and Deadlines
- User badges and permissions
- Categories for organizing surveys
